FEATURES

Course Authoring

Create full featured educative courses for your audience. Access all essential features like student management, lesson management and customisation.

Media

Use images, videos and PDF files to explain complex topics in a lucid fashion. You can even embed YouTube videos.

Payments

Make it all sustainable by offering paid content. CourseLit supports Stripe out of the box for payment processing. More payment methods are coming soon.

Blogs

You don't need a separate system to publish tutorials and blog posts. Using CourseLit, you can create posts right from its dashboard.

Open Source

You own the code. If you don't like something, change it to your liking. The code base is each to understand and it scales with your business.
